# Demand vs Availability Planner

## Overview
This project simulates how demand fluctuations and stock availability are evaluated in customer supply and order management roles.

It reflects real-world scenarios where customer demand exceeds available inventory and requires prioritization and escalation.

## Features
- Demand vs available stock comparison
- Identification of shortages
- Flagging of unrealistic delivery commitments
- Support for prioritization decisions

## Concepts Demonstrated
- Demand coordination
- Availability awareness
- Delivery feasibility
- Escalation readiness
- Data-driven decision-making

## Tools Used
- Microsoft Excel
- Basic operational logic

## Relevance
The model mirrors how ERP systems like SAP SD support availability checks and delivery confirmations at an operational level.
